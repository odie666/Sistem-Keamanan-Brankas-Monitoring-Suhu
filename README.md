# Sistem-Keamanan-Brankas-Monitoring-Suhu
Proyek ini adalah simulasi sistem keamanan brankas pintar yang dibangun menggunakan Arduino Uno di platform Tinkercad. Sistem ini mengintegrasikan kontrol akses fisik (Servo), sistem pendingin otomatis (Motor DC), dan monitoring kondisi lingkungan secara real-time (LCD &amp; Sensor Suhu).
